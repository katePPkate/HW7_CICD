## Описание проекта

В рамках домашнего задания реализован CI/CD-пайплайн для ML-сервиса, включающий:
- Обучение двух версий ML-модели (Stable и Canary)
- Развертывание с использованием стратегии **Canary Deployment**
- Настройку балансировщика Nginx для распределения трафика (90/10, 50/50, откат)
- Автоматизацию через GitHub Actions (GitVerse Actions)
